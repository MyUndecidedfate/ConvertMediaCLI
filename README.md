# ConvertMediaCLI
ConvertMediaCLI (CMCLI) is a Media format conversion CLI program that supports both Linux and Windows Systems
This is one of my first projects and note im not a good developer so this is coded in Python so expect it to be slower and eat up alot of space.
CMCLI uses both Pillow and FFMPEG libaries to handle the conversion. 

List of supported Media formats

Supported Image formats: JPG, JPEG, PNG, BMP, GIF, TIFF, TIF, WEBP, ICO, PPM, PGM, PBM

Supported Video formats: MP4, AVI, MKV, MOV, WMV, FLV, WEBM   

Supported Audio formats: MP3, WAV, AAC, FLAC, OGG, M4A  

Use --help for more instructions
